Function confirmMsg

  • confirmMsg(msg, btn?, title?): Promise<unknown>

  • Parameters

    • msg: string

      弹窗中显示的消息内容。

    • Optional btn: string[] = ...

      按钮的文字数组，默认包含两个按钮：“确定”和“取消”。

    • Optional title: string = '提示'

      弹窗的标题。

    Returns Promise<unknown>

    • 返回一个Promise，根据用户点击的按钮调用resolve或reject。

    Description

    显示确认消息框并根据用户选择返回Promise结果。

    此函数用于弹出一个确认消息框，包含自定义的消息、按钮文字以及标题。 根据用户点击的不同按钮，通过Promise异步地返回相应的处理结果。 支持最多6个自定义按钮，超出的按钮将不被处理。
