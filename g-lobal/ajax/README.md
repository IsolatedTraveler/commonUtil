# 打包说明

1. 可通过修改 var/ajax 中的部分参数更改 ajax 配置信息，具体配置如下：

| 参数名      | 说明                                                                                                             |
| :---------- | ---------------------------------------------------------------------------------------------------------------- |
| jqMode      | 设置不同的鉴权模式，现仅支持 magic:header 鉴权（该参数修改后，应在主程序同步提供鉴权方案）当 jqMode 为空，不鉴权 |
| contentType | 设置 ajax:post 提交数据方式(该参数修改后须同步修改主程序中的 dealAjaxData)                                       |
